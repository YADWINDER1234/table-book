## 🚀 Quick Command Reference

### Verification Scripts
```bash
# Windows
verify-setup.bat

# Mac/Linux
bash verify-setup.sh
```

### Development Scripts
```bash
# Windows
start-dev.bat

# Mac/Linux
bash start-dev.sh
```

### Manual Commands
```bash
# Backend setup
cd backend && npm install && npm run seed && npm run dev

# Frontend setup (separate terminal)
cd frontend && npm install && npm run dev
```
